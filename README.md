# EasyCopy Copier

Simplify your copying process by templates and many additional add-ons.

## Installation

### Windows
**For Windows users, your machine has to be Windows 10/11 in order for the application to run successfully.**

Download the executable in the releases area and follow the steps to complete the installation.

### GNU/Linux and MacOS
**Currently, the 'Run on startup' feature is only intended for Windows users. Do not try to change it in the settings.**

Run (provided you have [Git](https://git-scm.com) and [Python](https://python.org) installed):
```
> git clone https://github.com/Candyman-RDFZ/cdyCopy.git
> pip install -r requirements.txt
> python3 cdyCopy/main.py
```
